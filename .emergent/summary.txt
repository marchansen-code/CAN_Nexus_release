**what currently exists?**
The application is a full-stack knowledge management platform using React (frontend) and FastAPI (backend).
- **Authentication**: Custom JWT-based system with username/password. Admins manage users. Failed login attempts are logged for Fail2Ban. Users with active sessions are redirected from the login page to the dashboard.
- **Content Management**: A heavily enhanced TipTap-based editor allows creating articles with rich text, tables, images, and inter-article links ().
- **Article Features**: Articles support multiple categories (in a tree structure), tags, draft/published status, expiration dates, an important flag, group-based visibility, and a new comment section.
- **Editor Features**: The editor includes advanced table controls (creation dialog, resizing, cell coloring), image resizing, and article linking.
- **Recycle Bin**: Admins can view, restore, or permanently delete soft-deleted articles and documents from a dedicated Papierkorb page.
- **User/Group Management**: Admins can create users and groups to manage content visibility.
